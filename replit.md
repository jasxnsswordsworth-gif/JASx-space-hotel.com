# JASx Space Hotel Website

## Overview
A static single-page website for JASX (Journey Across Space eXploration) - a conceptual space tourism hotel project.

## Project Structure
- `index.html` - Main website file containing all HTML, CSS, and content
- No build system or dependencies required

## Development
The site is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured as a static site deployment with the root directory as the public folder.
